# 贡献指南

欢迎参与 AI Assistant Toolbox 项目！

## 如何贡献

### 1. Fork 仓库

点击页面右上角的 "Fork" 按钮，将仓库复制到你的 GitHub 账号。

### 2. Clone 到本地

```bash
git clone https://github.com/你的用户名/study.git
cd study
```

### 3. 创建分支

```bash
git checkout -b feature/你的功能名称
```

### 4. 开发和提交

```bash
# 修改代码后
git add .
git commit -m "feat: 添加xxx功能"
git push origin feature/你的功能名称
```

### 5. 提交 Pull Request

在 GitHub 上点击 "New Pull Request"，选择你的分支，填写说明后提交。

## 提交规范

请使用以下格式的 commit message：

- `feat: 新功能`
- `fix: 修复bug`
- `docs: 文档更新`
- `style: 代码格式调整`
- `refactor: 重构`
- `test: 测试相关`
- `chore: 构建/工具相关`

## 开发环境

- Java 17+
- Node.js 18+
- MySQL 8.0+

## 问题反馈

如有问题或建议，请提交 [Issue](https://github.com/fanruiqing/study/issues)。

感谢你的贡献！
